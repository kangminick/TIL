## CLI

### 커멘드 명령어들

- `ls` 폴더 내부의 파일 목록을 나열해준다.
- `pwd` 현재 폴더 경로 출력
- `mkdir`(폴더명) 폴더 생성
- `cd` (폴더명) 폴더 변경  ..은 상위폴더로 이동
- tab으로 자동완성 가능
- `git init` git으로 관리 시작
- `git status` git 상태 묻기
- `touch (파일명)` 파일생성
- `rm (파일명)` 파일 삭제
- `add`사진대에 올려 놓는다. ()
- `commit 사진을 찍는다.()`
